# TAREA-09ABRIL-ALGORITMOS-PRIMEROS-SIGUIENTES-PREDICION-
1. resumen de la tarea 
En esta tarea se desarrollo un analizador para gramáticas libres de contexto. El objetivo principal es procesar las reglas de una gramática y calcular tres conjuntos fundamentales para la construcción de compiladores los conjuntos primeros siguientes y de prediccion estos valores son esenciales para entender cómo una computadora debe leer y procesar un lenguaje de programación sin ambiguedades

2. Sobre el código y la solución
Se crearon dos codigos para demostrar el funcionamiento de los algoritmos con gramáticas diferentes en Python, lo que permite que la lógica sea limpia y fácil de seguir. El sistema recorre las reglas de forma repetitiva hasta que encuentra todos los símbolos terminales que pueden aparecer al inicio o después de cada variable, entregando al final una tabla detallada con los resultados.
